# **DocSaathi – AI Legal & Medical Document Simplifier**

**DocSaathi is a modern AI-powered web application designed to simplify complex legal and medical documents into easy-to-understand summaries.
The project focuses on clean UI, fast interaction, and accessible document understanding for normal users who struggle with technical language.**

## **Features**
### AI-powered document summary section
### Legal and medical document understanding concept
### Modern responsive landing page
### Smooth animations using Framer Motion
### Clean dark-themed UI
### Feature showcase cards
### Interactive buttons and transitions
### Example insight preview section
### Privacy-focused design approach
### Mobile responsive layout
