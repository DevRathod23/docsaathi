# DocSaathi

<div align="center">

AI-powered document intelligence platform that transforms complex Medical and Legal documents into simple, human-readable insights.

</div>

---

## 📌 Overview

DocSaathi helps users understand difficult documents without reading pages of technical language.

Upload a Medical report, Legal agreement, Insurance paper, or scanned document and the system automatically:

- Extracts text using OCR
- Analyzes document content using AI
- Detects important risks and key findings
- Simplifies complex terminology into plain language
- Generates structured summaries instantly

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/DevRathod23/docsaathi.git
```

### 2️⃣ Move Into Project

```bash
cd docsaathi
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Configure Environment Variables

Create a `.env.local` file:

```env
LLAMA_API_KEY=your_api_key_here
```

---

## ▶️ Run Locally

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
